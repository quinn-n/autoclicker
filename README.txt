Only tested on linux. Windows users beware.
autoclicker.py: a simple autoclicker, clicks 10 times per second by default. Or you can change pyautogui's delay.
Usage: autoclicker.py [delay]
Example: autoclicker.py .005

spawnAutoClick.py: A more complex autoclicker, uses multiple threads. Default number of threads is 10.
Usage: spawnAutoClick.py [Number of Threads]
